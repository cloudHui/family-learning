# 使用说明

## 登录

1. 打开管理员提供的“域名或 IP + 访问唯一路径”。
2. 初始管理员用户名为 `admin`，初始密码为 `123456`。
3. 普通用户第一次输入一个新用户名，密码框保持默认的 `123456`，即可建立个人档案。
4. 首次登录必须设置新密码；未修改前不能进入学习功能，下次登录仍会要求修改。
5. 管理员重置密码后，用户用 `123456` 登录并重新设置密码。

## 学习功能

- 首页只显示今日完成量、今日正确率、待复习错题和连续学习天数。
- 语文区支持浏览器免费中文语音、认字、听写以及手指/触控笔/鼠标书写。
- 数学区支持自定义范围和题量的加减法，自动判题并记录错题和用时。
- 错题连续答对 1/2/3 次后依次进入“复习中/基本掌握/已掌握”。
- 题目打印可以组合随机算术题和文字题，并选择是否显示答案。
- 资源中心支持 PDF、文本、图片、音频和视频；普通用户只读。
- 个人统计包含趋势、语文、数学、错题、学科分布、阶段进度和学习习惯。

## 管理后台

管理员可在学习首页右上角进入后台：

- 添加、停用、删除用户，分配功能权限和重置密码
- 查看全部用户、个人统计、当前页面和正在使用的功能
- 统计登录、访问页面、使用时长、设备、错误和高频错题
- 新增、编辑或删除汉字、学科内容、文字题模板、记录和错题
- 上传或删除不超过 50MB 的学习资源
- 预览或立即发送日报；系统每天 23:55（北京时间）检查，当天无人登录则跳过

在线数据每 10 秒更新，也可以点击主动刷新。

## 服务维护

```bash
sudo family-learning restart
family-learning status
family-learning logs
sudo family-learning update
```

Java 只监听本机 8088，公网访问统一经过 Nginx的 80/443。配置、学习数据与资源分别位于 `/etc/family-learning/` 和 `/var/lib/family-learning/`。
